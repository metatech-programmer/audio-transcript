## 🚀 DEPLOYMENT IN 3 STEPS

### Step 1: Dependencies (1 minute)

```bash
npm install
# Installs: Next.js, React, TypeScript, TailwindCSS, Zustand, etc.
```

### Step 2: Environment (2 minutes)

```bash
# Get API key from: https://console.groq.com/keys
# Then create .env.local with:
GROQ_API_KEY=gsk_your_key_here
GROQ_LLM_API_KEY=gsk_your_key_here
```

### Step 3: Deploy (2 minutes)

```bash
npm run dev          # Test locally (optional)
# OR
git push origin main # Triggers Vercel auto-deploy
# Visit: https://vercel.com/new to import
```

**Total: < 5 minutes from zero to production!**
