# Security Policy

If you discover a security vulnerability in this project, please do NOT open a public issue containing sensitive information (such as API keys or credentials).

Preferred reporting channels:

- Use GitHub Security Advisories (recommended): https://docs.github.com/en/code-security/security-advisories
- Or open a private/triaged issue and mark it clearly as a security report.

If you accidentally published secret keys in this repository, rotate them immediately with the provider and follow the provider's guidance.

Maintainers will aim to respond within 72 hours.
