# Quick Start - 5 Minute Setup

## TL;DR

```bash
git clone <repo>
cd record-audio
npm install

# Add to .env.local:
# GROQ_API_KEY=your_key_from_https://console.groq.com/keys

npm run dev
# Visit http://localhost:3000
```

## For Vercel Deployment

```bash
git push origin main
# Visit https://vercel.com/new
# Import repo → Add GROQ_API_KEY → Deploy
```

---

That's it! Your serverless lecture transcriber is ready.

## What You Get

✅ Record audio in browser
✅ AI transcription (Spanish + English)
✅ Auto-generate summaries
✅ Save & organize lectures
✅ Export to TXT/MD/JSON/Notion
✅ Fully serverless on Vercel
✅ No backend maintenance
✅ Mobile responsive
